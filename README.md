# Crypto Sentiment Trader Analysis

## Objective
Analyze the relationship between Bitcoin Fear & Greed sentiment and Hyperliquid trader performance.

## Dataset Summary
- Total Records: 211,218
- Unique Traders: 32
- Unique Coins: 246
- Common Trading Days: 479

## Key Findings
- Extreme Greed showed the highest average profitability.
- Win rates increased during optimistic market conditions.
- Fear periods showed larger average trade sizes.
- ANOVA confirmed statistically significant profitability differences across sentiment groups (p < 0.001).
